# sMRI_pipeline

Code for structural MRI analysis, including morphometric similarity network (MSN) construction, network statistics, clustering, and downstream clinical analyses.

## Repository structure

| Directory | Description |
|----------|-------------|
| `msn_pipeline` | MSN construction and feature extraction |
| `msn_stats` | Network statistical analyses |
| `clinical_analysis` | Clinical statistical analyses |
| `cluster_models` | Cluster-related machine learning models |
| `notebooks` | Analysis notebooks |
| `docs` | Documentation |

## Data

The dataset is available on **Zenodo** (DOI: **10.5281/zenodo.17989944**). Patient-level data are not included in this repository.
